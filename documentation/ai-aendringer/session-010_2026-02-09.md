# AI-ændringer — Session 010 (2026-02-09)

## Resumé
Kontaktsiden (kontakt.html) er implementeret med generel kontaktinfo (email + sociale medier) og polaroid-kort for de to co-founders. Ingen kontaktformular — siden er bevidst holdt ren og fokuseret.

## Ændrede filer
| Fil | Handling | Beskrivelse |
|-----|----------|-------------|
| `kontakt.html` | Oprettet | Komplet kontaktside med hero, kontaktinfo-sektion, team-kort og footer |
| `css/style.css` | Redigeret | Nye kontakt-specifikke CSS-klasser tilføjet |

## Detaljerede ændringer

### `kontakt.html` (oprettet)
- Hero-sektion med overskrift "Kontakt os" og velkomsttekst
- Generel kontaktinfo-sektion med gradient-baggrund: `info@stng-studios.dk` som pill-formet email-knap + sociale medier-ikoner (Instagram, TikTok, LinkedIn)
- Bølge-dividere mellem sektioner (genbrugt SVG-mønster)
- Team-kort sektion med to polaroid-kort (Simon Trinderup & Noah Grejsen) — genbruger eksisterende polaroid-design fra om-os.html
- Scroll-animationer på team-kort (slide-in-left / slide-in-right)
- Header, footer, Open Graph tags og ARIA-labels efter eksisterende mønster

### `css/style.css` (redigeret)
- Nye klasser: `.contact-hero`, `.contact-hero-text`, `.contact-general`, `.contact-email`, `.contact-social`, `.contact-team`, `.contact-cards`, `.wave-divider--flipped`
- Responsive regler i `@media (min-width: 768px)`: kontakt-hero padding og team-kort side-by-side layout

## AI's rolle
Simon var bekymret for at kontaktsiden ville se tom ud med kun to mails. Claude foreslog flere tilgange (kontaktformular, info + kort, alt sammen). Simon valgte "kun info + kort" — generel kontaktinfo med sociale medier og polaroid-kort for co-founders, uden kontaktformular. Claude implementerede siden med genbrugbare komponenter fra eksisterende sider.
