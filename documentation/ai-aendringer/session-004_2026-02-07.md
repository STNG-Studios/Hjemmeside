## Detaljerede ændringer

### `.claude/CLAUDE.md` (redigeret)
- Tilføjet ny sektion "Clarifying Questions (OBLIGATORISK)" efter "Sprog"-sektionen (linje 32-39)
- Reglen kræver at Claude stiller afklarende spørgsmål før implementering

### `css/style.css` (oprettet)
- CSS custom properties med farveskema baseret på hero-banner (mørk navy/sort med blå-lilla accenter)
- Global reset og base-styling
- Sticky navbar: transparent → solid ved scroll, hamburger-menu på mobil, horisontal menu på desktop
- Hero-sektion: full-viewport baggrundsbillede med gradient-overlay og centreret tekst
- App-sektion: billedkarrusel med squircle-corners (progressiv enhancement), dots-navigation
- Founder's note: placeholder-billede med tekst og blockquote-styling
- Footer: tre-kolonne layout med links, selskabsinfo og social media ikoner
- Scroll-animationer: slide-in-left/right klasser (kun aktiveret på desktop ≥1024px)
- Responsive breakpoints: 768px (tablet) og 1024px (desktop)

### `index.html` (oprettet)
- HTML5 med `lang="da"` og Open Graph prefix
- `<head>`: charset, viewport, title, meta description, Open Graph tags (title, type, url, image, description, site_name, locale), Schema markup (Organization JSON-LD med adresse, CVR, kontaktpunkt), Google Fonts (Inter), CSS-link
- `<header>`: sticky navbar med logo (link til forside), hamburger-knap, nav-menu (Om os, Apps, Kontakt)
- `<section class="hero">`: hero-banner baggrund, h1 "STNG STUDIOS", placeholder undertekst
- `<section class="app-section">`: overskrift, karrusel med 3 app-screenshots (gym, track, lib), dots-navigation, beskrivende tekst
- `<section class="founder-section">`: placeholder-billede (SVG person-ikon), blockquote med stifter-tekst
- `<footer>`: tre kolonner — links (FAQ, Servicevilkår, Om os), selskabsinfo (CVR, adresse), social media (mail, Instagram, TikTok, LinkedIn med SVG-ikoner)
- jQuery 3.7.1 CDN + link til main.js

### `js/main.js` (oprettet)
- Sticky navbar: tilføjer `.scrolled` klasse ved scroll > 50px
- Hamburger-menu: toggle `.active` klasse, aria-expanded, body scroll-lock, overlay-klik lukker menu
- Billedkarrusel: auto-rotation hvert 4. sekund, dots-navigation, touch/swipe support, pause ved hover
- Scroll-animationer: IntersectionObserver tilføjer `.visible` klasse til slide-in elementer (kun desktop ≥1024px)

## AI's rolle
Simon bad om en komplet startside med specifikt layout (header, hero, app-sektion med karrusel, founder's note, footer). Claude stillede afklarende spørgsmål om app-indhold, farveskema, karrusel-opførsel, navbar-stil og kontaktinfo. Claude implementerede derefter designet med mobile-first tilgang, vurderede farveskema ud fra logo og hero-banner, og valgte tekniske løsninger som IntersectionObserver for scroll-animationer og jQuery for karrusel/menu-interaktivitet.
