## Ændringer

### 1. `js/main.js` — IntersectionObserver (linje 142-166)

**Nuværende adfærd:**
- Observer tilføjer `.visible` når element er i viewport
- `observer.unobserve(entry.target)` fjerner overvågning → animation kører kun én gang

**Ny adfærd:**
- Fjern `observer.unobserve()` helt
- Når `entry.isIntersecting` er `true` → tilføj `.visible`
- Når `entry.isIntersecting` er `false` → fjern `.visible`
- Animationen nulstilles automatisk hver gang elementet forlader viewport

```js
var observer = new IntersectionObserver(function (entries) {
  entries.forEach(function (entry) {
    if (entry.isIntersecting) {
      $(entry.target).addClass('visible');
    } else {
      $(entry.target).removeClass('visible');
    }
  });
}, observerOptions);
```

### 2. `css/style.css` — Easing-kurve (linje 588-605)

**Nuværende:** `transition: opacity 0.8s ease, transform 0.8s ease`
**Ny:** `transition: opacity 0.8s ease-in-out, transform 0.8s ease-in-out`

Ændr begge `.slide-in-left` og `.slide-in-right` inden for `@media (min-width: 1024px)`.
