## Ændring: `css/style.css`

**Nuværende:**
```css
.wave-divider {
  background-color: #1a1a3e;
  line-height: 0;
  margin-top: -1px;
}

.wave-divider svg {
  display: block;
  width: 100%;
  height: 80px;
}
```

**Ny:**
```css
.wave-divider {
  line-height: 0;
  position: relative;
  z-index: 1;
  margin-bottom: -80px;
}

.wave-divider svg {
  display: block;
  width: 100%;
  height: 80px;
}
```

**Hvad sker der:**
- Baggrundsfarven fjernes → ingen farve lyser igennem under bølgekurverne
- `margin-bottom: -80px` trækker founder-sektionen 80px op, så dens gradient vises direkte igennem de åbne områder under SVG-bølgen
- `position: relative; z-index: 1` sikrer at SVG'en (med `#0a0a12`) ligger OVER founder-sektionens gradient
- Resultat: den mørke app-farve har en bølget underkant → sømløs overgang til gradient

**NB:** Simon har midlertidigt ændret SVG fill-farven til test. Denne skal sættes tilbage til `#0a0a12` (app-sektionens baggrund).
